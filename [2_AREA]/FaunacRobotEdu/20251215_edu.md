# 1. Faunac Robot  소개
## 1.1. 역사
- 50년대 CNC로 시작
- 2000년에 R-2000i (톤 단위 payload 모델) 히트
- 2003년에 로봇이 로봇을 만들기 시작
- Delta 로봇의 등장
- Green Robot(협동 로봇) 등장

## 1.2. 안전 유의 사항 
- Door와 Interlock 
- 2인 1조로 작업
- __3 Mode Switch__ 인지
	- __T1__: Test Mode1, 속도: < 250 \[mm/s\] 
	- __T2__: Test Mode2, 속도: 100% (수치는 override 가능)
	- __Auto__: 자동 운전

## 1.3. 로봇 구성
1. Manipulator
2. Actuator
3. Sensor: 
4. Controller
5. I/O Controller
6. Axes Control Module
7. Power Supply(갑작스럽게 전원이 내려갈 수 있는 환경에서 구축하는게 좋음)
8. UOP: 외부와  Robot과 주고 받는 신호
9. Jig & Fixture

## 1.4. 로봇 모델명

> '기종' / 'payload' + 'type'

__type__: 'F', 'L', 'R', 'U', 'S'
F: 기본 모델
L: Long Arm
R: Rack Mount, 높은 위치에 설치되어 작업 영역이 base 밑에 있는 타입
U: Upside-down, 천장에 설치되어 있는 타입
S: Short Arm

## 1.5. Controller 모델 (i : Intelligence)
- R-30iA
- R-30iB
- R-30iB Plus (2018년 이후 모델)

	 ___intelligence 특징 3가지___
	1. vision sensor, force sensor  부착 가능, Soft Float 기능 (충격 흡수 기능)
	2. Networking 기능: 산업용 Ethernet 기능 강화
	3. Robotization: 자동화 

## 1.6. Controller 기본
- Type
	- Mate Type
	- Cabinet A Type
	- Cabinet B Type(확장 가능)
- SOP (Standard Operating Panel)
	- Fault Reset, Cycle Start, EMG 버튼 등등의 기본 기능 인터페이스
- 메모리
	- FROM: Maker 영역 메모리
	- SRAM: User 영역 메노리 (* 휘발하기 때문에 DC3V 공급 해줘야함. Maker 통해서 4년단위 교체 권장. 백업을 위해 전원 키고 30초 ~ 1분 On 후 전원 Off 하고 백업)
- 원점 백업용 배터리
	- DC6V(1.5V 알카리인 4개, 시중에서 구매)
	- 전원이 Zero가 되면 다시 Mastering을 해줘야 하기 때문에, Low가 될때  UOP 통해서 알림이 뜨도록 권장. 

#  2. Pendant

## 2.1. 3 Switch Button

1. 유효 스위치
	- Auto <-> T1 <-> T2
2. 데드맨 스위치

	- 유효 스위치와 상관 없이 비상 정지

## 2.2. 기능키

### 2.2.1.  SHIFT

- 팬던트의 키 중에 파란색 배경의 키를 동작시키기 위한 키.
- 먼저 SHIFT 키를 누를 상태에서 파란색 배경의 키를 누르면 나중에 누른 키.

###  2.2.2. F1 ~ F5, Next 

- 화면 맨 아래의 메뉴를 '서브 메뉴' 명명.
- F1 ~ F5는 서브 메뉴의 맵 키(버튼)
- Next는 서브 메뉴 맨 오른쪽에 '>' 버튼의 키 
- 서브 메뉴에 \[]  대 괄호로 묶여 있는건 눌렀을때 더 선택할 것들을 Context Menu로 보여줌.

### 2.2.3. PREV

- 현재 화면에서 추가로 보여주고 있는 팝업 창, 부가 표시 정보를 닫음.
- 이전 화면으로 가는 키는 아님.

### 2.2.4. SELECT, EDIT, DATA 단축키

- __SELECT__: 화면에 프로그램 리스트를 보여줌.
- __EDIT__: 화면에 마지막에 선택한 프로그램의 코드를 보여줌.
- __DATA__: 화면에 Register 값들을 보여줌. 여러 Register 종류중 마지막에 설정한 페이지를 보여줌. F1의 Type으로 종류 변경 가능.

### 2.2.5. DISP

- 화면 분할 및 분활 화면 이동
- Shift 키와 조함으로 사용.

### 2.2.6. FCTN

-  function 키
- 현재 화면서에서 부가 메뉴를 팝업 시킴.

### 2.2.7. DIAG
- 진단 창(SHIFT)을 보여줌. Error 같은 내용을 보여줌. 

### 2.2.8. ENTER

- 커서를 이동하여 입력할 부분에서 입력을 끝낸 후, 항상 Enter를 눌러야 완료가 된다.
